# Yoga: A Forgotten Discipline

**Reviving ancient wisdom through modern technology**

---

## Overview

**Yoga: A Forgotten Discipline** is a web platform designed to reconnect people with the true essence of yoga. It combines traditional knowledge with computer vision to provide **real-time posture correction** and **guided practice**.

At the heart of this system is **Bodhi**, an intelligent posture guidance system that observes your movements and helps you improve your yoga practice instantly.

---

## Meet Bodhi — Your Yoga Companion

**Bodhi** is a real-time posture correction system built using **MediaPipe**. It acts like a virtual yoga instructor by:

* 📸 Tracking your body movements via webcam  
* 🧍 Detecting posture using pose estimation  
* ⚠️ Identifying incorrect alignments  
* ✅ Providing instant feedback to improve your form  

This allows users to practice yoga **safely and effectively**, even without a physical instructor.
